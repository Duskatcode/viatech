# Vitatech Maintenance Platform

Plataforma web para gestionar equipos biomédicos, hojas de vida, mantenimientos preventivos/correctivos, evidencias y reportes.

## Stack definido

- Backend: NestJS + TypeScript
- Frontend: React + Vite + TypeScript
- UI: Tailwind CSS + shadcn/ui
- Base de datos: PostgreSQL
- ORM: Prisma
- Auth: JWT + Refresh Token
- API Docs: Swagger/OpenAPI
- Deploy Frontend: Vercel
- Deploy Backend: Render
- DB Cloud: Supabase PostgreSQL

## Flujo principal del MVP

Empresa → Sede → Área → Equipo → Orden de mantenimiento → Historial → Reporte

## Fases

- Fase 0: definición técnica, dominio, reglas y contratos.
- Fase 1: estructura monorepo.
- Fase 2: backend NestJS.
- Fase 3: frontend React.
- Fase 4: base de datos y Prisma.
- Fase 5: módulos del dominio.
- Fase 6: reportes, dashboard y despliegue.
