# 00 — Alcance del Proyecto

## Nombre temporal

Biomed Maintenance Platform

## Objetivo

Construir una plataforma web para gestionar equipos biomédicos, hojas de vida, mantenimientos preventivos/correctivos, evidencias y reportes.

## Problema

Muchas organizaciones gestionan equipos biomédicos usando hojas de cálculo, carpetas físicas, documentos dispersos o formatos manuales.

Esto genera problemas como:

- Dificultad para consultar el estado real de los equipos.
- Pérdida de historial técnico.
- Falta de trazabilidad.
- Reportes lentos.
- Riesgo de vencimiento de mantenimientos preventivos.
- Baja visibilidad sobre sedes, áreas y responsables.

## Solución

Crear una plataforma centralizada donde cada equipo tenga:

- Identificación.
- Ubicación.
- Estado.
- Hoja de vida.
- Historial de mantenimiento.
- Evidencias.
- Reportes.

## Incluido en MVP 1

- Login.

- Hoja de vida del equipo.

- Reportes básicos.
